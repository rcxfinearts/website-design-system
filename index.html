Element Styles:

H1 Elements

Plus Jakarta Sans is the font type being utilized for the H1 header on pages. Variations of this font will be used for links in italic type and on buttons in extra bold type.

H2 Elements

Dancing Script is the font type being utilized for the H2 headers in sections on pages. Variations of this font will be bold for headers the eye should be drawn to.

Paragraphs

EB Garamond is the font type being utilized for paragraph elements. Variations of this font will include bold for words that are more important.
